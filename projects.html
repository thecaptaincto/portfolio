NeonChat

Privacy-first messaging with aesthetic soul

Overview

NeonChat is a real-time messaging application built for communities who value both privacy and aesthetics. It features end-to-end encryption, custom themes inspired by East Asian night aesthetics, and a lantern-inspired UI that makes secure communication feel warm and inviting.

The project started because I noticed most secure messaging apps feel cold and corporate. I wanted to prove that privacy doesn't have to sacrifice beauty or user experience.

Key Features

  • End-to-end encryption for all messages
  • Custom theme system with lantern-inspired designs
  • Real-time messaging with WebSocket connections
  • Group chat with role-based permissions
  • File sharing with encryption
  • Responsive design that works beautifully on all devices

Technical Stack

Frontend: React, Redux, Socket.io Client
Backend: Node.js, Express, Socket.io
Database: MongoDB with Mongoose
Security: AES-256 encryption, JWT auth

Challenges & Solutions

The biggest challenge was implementing end-to-end encryption while maintaining real-time performance. I solved this by implementing a hybrid encryption system that uses symmetric encryption for messages and asymmetric encryption for key exchange.

Weather forecasts as visual poetry

Heritage Archive

Preserving stories for future generations

Overview

Heritage Archive is a digital preservation platform for cultural stories, historical photographs, and family memories. It's built to honor the past while making it accessible and engaging for future generations through modern technology.

Key Features

  • Upload and organize historical photos with metadata
  • Timeline view for chronological storytelling
  • OCR for extracting text from old documents
  • Collaborative editing and annotation
  • Export collections as beautiful PDFs or web galleries
  • Works offline as a Progressive Web App

Technical Stack

Frontend: Vue.js, Vuex
Backend: Firebase (Auth, Firestore, Storage)
Image Processing: Sharp, Tesseract.js
PWA: Service Workers, IndexedDB

LanternTimer

Focus illuminated

Overview

LanternTimer is a minimalist productivity timer that combines the Pomodoro technique with calming visual design. As you work, a virtual lantern glows softly, dimming during breaks to signal transition. It's designed for deep work sessions with minimal distraction.

Key Features

  • Customizable work/break intervals
  • Ambient lantern glow that responds to your focus state
  • Optional background sounds (rain, cafe, silence)
  • Session statistics and productivity insights
  • Minimal, distraction-free interface

Technical Stack

Frontend: Vanilla JavaScript
Animation: Web Animations API, CSS
Storage: Local Storage
Audio: Web Audio API
